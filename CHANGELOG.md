# Changelog

## 0.1.0

- Peek into `.py` module source directly from notebook `import` statements via CodeLens
- Export notebook definitions (functions, classes, assignments) to `.py` modules
- Export imports to target modules while keeping them in the notebook
- Auto-save peeked files when switching editors or executing cells
- Autoreload prompt on notebook open
- CodeLens filtering: only show peek buttons for local/editable packages
